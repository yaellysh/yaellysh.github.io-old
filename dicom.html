Project Summary

I recently completed a complex project involving the processing and transformation of ultrasound image data and its conversion into 3D DICOM files. Initially, the ultrasound data was provided as a single 416x3840 image containing 80 individual frames, each formatted in the Cartesian coordinate system. However, traditional ultrasound images are typically in a sector scan format, which requires a polar transformation.

To address this, I began by meticulously splicing out each of the 80 frames from the combined image, ensuring precision in the separation process to maintain the integrity of the data. Each frame was then carefully rotated to align it correctly before applying a polar transformation to convert it from its original Cartesian form into the sector scan shape, which is the standard in conventional ultrasound imaging. This transformation was crucial for accurately representing the anatomical structures in a format recognizable in medical diagnostics. Additionally, the original image quality presented a challenge, as it was insufficient for producing high-resolution outputs. To overcome this, I employed advanced interpolation techniques to enlarge the frames, enhancing their resolution. This process not only improved the clarity and detail of the images but also ensured that the final ultrasound outputs were free of artifacts and noise, making them suitable for clinical use.

Building on this foundational work, I developed an automated pipeline to convert the processed 2D ultrasound frames into a cohesive 3D DICOM file. This involved implementing algorithms for precise spatial alignment of the slices, ensuring that each frame was correctly positioned relative to the others to accurately reconstruct the volumetric data. Volumetric reconstruction was then performed, combining the aligned slices into a comprehensive 3D model. I optimized the entire process to efficiently handle large datasets, minimizing processing time without sacrificing accuracy. To validate the integrity and accuracy of the final 3D DICOM file, I used industry-standard DICOM viewers, including Weasis, 3D Slicer, and OsiriX, confirming that the reconstructed images met the required clinical standards for diagnostic purposes. This project not only involved the transformation of ultrasound data but also contributed to the creation of high-quality 3D DICOM files, which are critical for accurate medical imaging and diagnosis.

My goal working on this project was to create a proof of concept, showing that this could in fact be done and proving that this was a project that was worth more time and investment. While this is the extent of the work I did regarding this, the final goal was to create a true to life, accurate 3D model/ DICOM file from which various measurements could be taken. The data I used was primarily dummy data (i.e. the size of the model I created was not the same as the actual scan) but I was able to demonstrate that this goal was attainable.
