Internship Projects

DICOM Image Conversion (Summer 2024)

I developed a pipeline in Python to convert sequences of 2D Cartesian images into 3D DICOM files, enhancing their use in medical imaging applications. I implemented algorithms for precise spatial alignment and volumetric reconstruction to ensure the 3D models were accurate and high-quality. The process was continuously optimized to handle large datasets efficiently while maintaining data integrity and compliance with medical imaging standards. I also worked with DICOM viewers such as Weasis, 3D Slicer, and OsiriX to validate and visualize the generated 3D DICOM files.

Diagnosis Classification (Summer 2023)

I worked on a project where I transformed diagnosis text files into structured JSON, making data easier to manage and store. I also developed a Python interface that allowed healthcare professionals to label conditions across more than half a million diagnosis documents. To support AI model training, I created a large dataset from unstructured text for extracting and labeling diagnoses.

Academic Projects

Assembly Tetris

I completed a project where I used MIPS assembly to create a version of the classic Tetris game. Since we didn't have access to physical MIPS hardware, I simulated the game using tools like Saturn or MARS. These simulators allowed me to not only work with the MIPS processor but also to simulate a Bitmap Display and Keyboard input, bringing the game to life in a virtual environment.

Tierlist Creator

I worked on an application that lets users organize items into tier lists based on different categories. Users can either choose from existing categories, create their own custom tier lists, or generate random ones. Users can also create accounts to save their tier lists and access them later. To add more personalization, we used the ChatGPT OpenAI API to help users create tournaments by generating lists of relevant items.
